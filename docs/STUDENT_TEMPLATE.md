# Шаблон структуры для студента

Каждый студент должен скопировать эту структуру в свою папку `works/<ТВ-ЛОГИН>/`.

## Структура

```
works/ivan_puplin/
├── README.md                    # Описание студента (опционально)
├── labN/
│   ├── README.md
│   ├── Makefile
│   ├── src/
│   │   ├── main.cu
│   │   └── shapes.cu
│   └── include/
│       └── shapes.h
```

## Шаблон README для Lab 1

```markdown
# Lab 1: CUDA Graphics

## Описание

Программа выводит в терминал геометрическую фигуру (квадрат, круг, треугольник),
используя параллельные вычисления на CUDA.

## Требования

- CUDA Toolkit 11.0 или выше
- GPU с compute capability 7.0+
- Linux с GCC 9.0+
- Make

## Компиляция

```bash
make
```

## Запуск

```bash
./graphics [size] [shape]

# Примеры:
./graphics 20 circle
./graphics 15 square
./graphics 15 triangle
```

## Результаты

```
Размер: 20
Фигура: круг

        ****
      **    **
    **        **
   **          **
  **            **
  **            **
   **          **
    **        **
      **    **
        ****
```

## Особенности

- Используется shared memory для оптимизации
- Каждый поток рассчитывает одну позицию
- Линеаризация 2D сетки потоков

## Замечания

- Максимальный размер: 200 (из-за лимита потоков на блок)
- Для большего размера нужно использовать несколько блоков
